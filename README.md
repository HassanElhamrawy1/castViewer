# castViewer
 
Mini TeamViewer MVP built with Qt6 C++  
Allows screen sharing and remote control (prototype) over LAN and internet.
 
## Features
 
- Capture primary screen using `QScreen::grabWindow()`
- Display screen in Qt Widgets window
- Update screen every second (adjustable)
- Initial setup for sending screen via TCP/SSL (future step)
